Detailed Description

Classe que representa um fluxo exponencial em um sistema.

A classe FlowExponential é uma subclasse de Flow e representa um fluxo exponencial em um modelo de sistema. Ela herda construtores da classe Flow e implementa o método execute para calcular o valor do fluxo exponencial.
