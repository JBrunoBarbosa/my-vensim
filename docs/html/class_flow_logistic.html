Detailed Description

Classe que representa um fluxo logístico em um sistema.

A classe FlowLogistic é uma subclasse de Flow e representa um fluxo logístico em um modelo de sistema. Ela herda construtores da classe Flow e implementa o método execute para calcular o valor do fluxo logístico.
