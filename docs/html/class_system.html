Detailed Description

Classe que representa um sistema puramente virtual.

A classe System é responsável por representar um sistema em um modelo. Ela armazena um nome descritivo e um valor associado ao sistema. Essa classe fornece métodos para definir e obter o nome e o valor do sistema.
